AdvancedGroupService

Deprecated
NotReplicated

This item is not replicated across Roblox’s server/client boundary.

AdvancedGroupService is a service that allows developers to fetch information about a Roblox group from within a game with more accessibility than the GroupService class..

The class has a number of useful applications, such as detecting if a player is an ally or enemy upon joining the game without much coding.
